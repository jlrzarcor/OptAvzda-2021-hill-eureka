************
Installation
************


Python Package
==============

Install package with:
  
.. code-block:: bash

   #execute next line being in `src` dir where `setup.py` file is 
   #after cloning repo:
   #https://github.com/jlrzarcor/OptAvzda-2021-hill-eureka
   pip install .

or in editable mode:

.. code-block:: bash

   #execute next line being in `src` dir where `setup.py` file is 
   #after cloning repo:
   #https://github.com/jlrzarcor/OptAvzda-2021-hill-eureka
   pip install -e .

or with url:

.. code-block:: bash

    pip install "git+https://github.com/jlrzarcor/OptAvzda-2021-hill-eureka#egg=hill_cg&subdirectory=src"
